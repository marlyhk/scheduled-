# Scheduled — Vercel Ready Website

This is the free Vercel-ready version of your Scheduled tutoring website.

## What this version includes
- Admin / tutor / student / group demo logins
- Booking system
- Tutor hourly rates
- Individual and group pricing
- Whish / Cash payment logic
- WhatsApp booking confirmation
- Tutor financial tab
- Student payment history
- Tutor availability
- Documents demo section
- White + baby blue Scheduled design

## Important
This version stores data in the browser using localStorage.
That means it is good for demo/testing, but not yet a secure real database.

For real student/tutor accounts and real file uploads, connect Firebase later.

## How to publish for free on Vercel
1. Create a GitHub account.
2. Create a Vercel account.
3. Create a new GitHub repository named scheduled.
4. Upload all files in this folder to that repository.
5. On Vercel, click Add New Project.
6. Import the scheduled GitHub repository.
7. Click Deploy.
8. Your website will become live on a free Vercel link.
